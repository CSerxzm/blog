# blog
该项目实现的是blog，用于展示自己的一些文章和简历，喜欢的可以star。
## 涉及的技术环境：
#### 环境
* jdk 1.8
* IDEA 2017
* MAVEN 3.6.0
#### 技术
* semantic ui
* spring boot
* mybatis
* redis
## 功能
* blog的管理，发布，如草稿
* 评论的管理，以及评论发送到邮件
* blog的赞赏（扫描支付宝，微信转账，喜欢的可以转个三元五元，支持服务器的运行）
